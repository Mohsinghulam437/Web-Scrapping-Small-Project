# Web Scraping Quotes Project

This is a simple Python web scraping project where I collected quotes from a website and processed the data.

## 🚀 What I did
- Scraped multiple pages using requests
- Parsed HTML using BeautifulSoup
- Stored raw pages locally
- Extracted quotes and authors
- Filtered quotes based on specific tags (e.g., "life")
- Cleaned and structured data using pandas
- Exported final dataset to CSV

## 🛠️ Technologies Used
- Python
- requests
- BeautifulSoup
- pandas

## 📂 Output
The final dataset is saved as:
`life_quotes.csv`

## 📌 Learning Outcome
This project helped me understand how data is collected, cleaned, and prepared before using it in AI/ML models.
